Missing Gap in AI Tools

There exist tools that can collect data from edge devices (AWS IOT Core, IBM Watson, etc.) and siloed controllers that work well on single edge devices (IoTium, Ericsson, etc.). There is a missing gap that represents the communication of critical information across smart devices to improve system-wide performance.

The Solution: Synaptik

Synaptik is a decentralized self-learned M2M communication tool that allows an IOT enabled endpoint to communicate instructions to neighboring endpoints to maximize an system performance. Rather than having human middle-managers to orchestrate and direct systems in real time, intelligent machines can automatically identify opportunities to coordinate with each-other. Our AIaaS framework integrates seamlessly on top of existing control systems.

How It Works?

This tool allows users to describe the performance metrics they want to optimize and specify what controls the system has over each machine's behavior. Then, the software learns a communication protocol along each edge between different classes of machines. Once a communication protocol between any group of machines is learned, then we can reuse that learned protocol whenever a new connection is made. The communication protocol can then be used to output a recommended strategy that can be used to enhance existing controllers.
